
# NeighborFit (Without MongoDB)

NeighborFit is a full-stack web app that recommends the best neighborhood for you based on lifestyle preferences.

## Tech Stack
- React (frontend)
- Node.js + Express (backend)

## How to Run

### Backend:
```bash
cd server
npm install
npm start
```

### Frontend:
```bash
cd client
npm install
npm start
```
